# AndroidWiFiConnect
Android Native kotlin lifecycle observer, load a list of wifi to connect into every single one

Your json should be like this:
```json
[
    {
      'ssid': 'ssid1',
      'password': 'password123',
      'security': 'WPA2'
    },
    {
      'ssid': 'ssid2',
      'password': 'password123',
      'security': 'WPA'
    },
    {
      'ssid': 'ssid3',
      'password': 'password123',
      'security': 'WPE'
    }
]
```
